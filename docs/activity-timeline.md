# Activity Timeline & Audit Logs UI

## Overview
The **Activity Timeline** (`/dashboard/copilot/actions`) provides a complete audit history of AI task executions.

## Features
- **Execution Table**: Displays status badge (`Success`, `Failed`, `RequiresConfirmation`), original command text, invoked tool name, execution timestamp, and result summary.
- **Status & Tool Filtering**: Easily search or filter task logs by execution outcome.
- **Re-run Actions**: Allows business owners to re-execute any previous command with one click.
