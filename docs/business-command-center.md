# Business Command Center UI

## Overview
The **Business Command Center** (`CommandCenter.tsx`) renders categorized quick-action prompt cards inside the Copilot workspace.

## Command Categories
1. **Sales**: "Show today's revenue", "Assign new leads to Rahul", "Show dashboard".
2. **Customers**: "Which customers haven't been contacted?", "Show pending tasks".
3. **Finance**: "Create invoice for $500", "Show unpaid invoices".
4. **Marketing**: "Create QR Code for my landing page", "Send WhatsApp message to client", "Send payment reminder email".
5. **Operations**: "Generate this week's report", "Run lead nurture workflow".

## User Experience
- One-click selection immediately executes the selected prompt or populates the chat input bar.
- Collapsible workspace header toggle allows users to expand or hide shortcut cards.
