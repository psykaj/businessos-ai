# SEO Module

The SEO Module allows organizations to optimize their public-facing White Label Platform assets for search engines.

## Features
- **Meta Tags**: Title, Description, and Keywords.
- **Social Sharing**: Open Graph and Twitter Cards.
- **Crawlers**: Robots indexing directives and Canonical URLs.

## API Endpoints
- `GET /api/seo` - Retrieve the SEO configuration for the organization.
- `PUT /api/seo` - Update SEO settings.
