# Reports Center & Export UI Architecture

The **Reports Center** (`/dashboard/reports`) and **Export Center** (`/dashboard/export-center`) provide automated report generation and multi-format document delivery.

## Capabilities
- **Report Categories**: Executive, Sales, Marketing, CRM, Finance, Workflow, AI Usage.
- **Output Formats**: Formatted PDF briefs, Excel (XLS) spreadsheets, and raw CSV streams.
- **Interactive Report Preview Modal**: Preview report summaries directly in the browser before exporting.
- **Instant File Downloads**: Triggers native browser download streams for PDF, XLS, and CSV files.
