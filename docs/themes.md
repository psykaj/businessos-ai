# Theme Engine

The Theme Engine allows deep customization of UI elements across the SaaS and Landing Pages.

## Features
- **JSON Configurations**: Store extensive style definitions (Border radius, shadows, card styles, component variants).
- **Modes**: Light, Dark, System modes.
- **Multiple Themes**: Orgs can create multiple themes and switch default active theme.

## API Endpoints
- `GET /api/themes`
- `GET /api/themes/{id}`
- `POST /api/themes`
- `PUT /api/themes/{id}`
- `PUT /api/themes/{id}/default`
- `DELETE /api/themes/{id}`
