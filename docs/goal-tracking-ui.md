# Goal Tracking UI Architecture

The **Goal Management Page** (`/dashboard/goals`) enables business leaders to set quantitative targets and monitor milestone completion.

## Features
- **Goal Creation Modal**: Create custom goals (Revenue Target, Lead Volume, Customer Signups, Subscription Renewals).
- **Progress Visualizations**: Animated progress bars showing percentage completion (`CurrentValue / TargetValue * 100`).
- **Live KPI Synchronization**: Sync goal progress automatically against calculated backend KPIs.
- **Status Badges**: Visual indicators for `InProgress`, `Achieved`, `Behind`, and `Failed`.
